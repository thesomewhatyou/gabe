# Welcome to Gabe Docs

Gabe is a multifunctional Discord bot packed with moderation tools, fun commands, and customization options. This documentation covers everything you need to install, configure, and extend Gabe.

## Quick Links

- [Setup](./setup/) — Install dependencies, configure environment variables, and launch the bot.
- [Commands](./commands/) — Browse the full catalog of moderation and fun commands.
- [Configuration](./config/) — Reference for environment variables and JSON files.
- [Containers](./containers/) — Run Gabe with Docker or Podman.
- [PostgreSQL](./postgresql/) — Configure Gabe to use PostgreSQL instead of SQLite.
- [Custom Commands](./custom-commands/) — Build your own commands with the command framework.
- [Runtime Compatibility](./runtimes/) — Notes on running Gabe with Node.js, Deno, or Bun.
